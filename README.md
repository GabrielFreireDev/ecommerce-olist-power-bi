# Olist E-commerce Analytics

## Visão geral
Projeto de análise descritiva utilizando dados públicos da Olist (Kaggle),
com foco em métricas de vendas, clientes, vendedores, avaliações e entregas.

## Ferramentas
- Power BI
- Power Query (M)
- DAX
- Figma

## Estrutura do projeto
- Modelagem em estrela
- Medidas DAX documentadas externamente
- Dashboard interativo

## Observações
- Dados públicos disponibilizados na plataforma <a href="https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce" target="_blank" rel="noopener">kaggle</a>
- Projeto para fins educacionais e portfólio profissional
